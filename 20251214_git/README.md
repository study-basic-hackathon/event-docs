# PDFに変換
marp slides.md -o slides.pdf --allow-local-files

## 📚 各資料の使い方

### 運営メンバー向け

1. **`plan.md`** を読んで全体像を把握
2. **`setup_guide_for_organizers.md`** で事前準備
3. **`slides.md`** で当日のプレゼンテーション
4. 必要に応じて **`practical_exercises.md`** を追加演習として実施

### 参加者向け

1. 事前に **`preparation.md`** で準備
2. 当日は **`handson_guide.md`** を見ながら実習
3. 後日復習用としても利用可能

## 💡 カスタマイズのヒント

### 時間調整

- ハンズオンの時間は参加者のレベルに応じて調整可能
- 実践演習は時間があれば実施（省略してもOK）
- 質疑応答の時間を多めに取ることも検討

### 内容の追加・削除

- **初心者が多い場合**: 環境構築に時間をかける
- **経験者が多い場合**: `practical_exercises.md` の演習を追加
- **時間が足りない場合**: ブランチ操作を簡略化

### グループ分け

- **4人未満**: グループ分けなしで全体進行
- **4-8人**: 2グループ
- **9-12人**: 3グループ
- **13-16人**: 4グループ
