# 🚀 プログラミング初学者の会 - メンバー紹介サイト

Gitハンズオンセミナー 2025/12/14 の実践演習用リポジトリです。

## 📖 プロジェクト概要

このプロジェクトは、Gitハンズオンセミナーの参加者が実際のチーム開発フローを体験するためのWebサイトです。

参加者一人ひとりが自分のプロフィールカードを追加し、最終的に全員の紹介ページが完成します。

## 🎯 学習目標

このプロジェクトを通じて、以下のスキルを習得します：

- ✅ リポジトリのクローン
- ✅ ブランチの作成と切り替え
- ✅ ファイルの編集とコミット
- ✅ リモートリポジトリへのプッシュ
- ✅ プルリクエストの作成
- ✅ チーム開発のワークフロー

## 📁 プロジェクト構成

```
community-site/
├── index.html          # メインページ
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── members.js     # メンバーデータ（ここを編集します）
├── README.md          # このファイル
└── MEMBER_GUIDE.md    # メンバー追加ガイド
```

## 🚀 クイックスタート

### 1. リポジトリをクローン

```bash
git clone https://github.com/study-basic-hackathon/community-site.git
cd community-site
```

### 2. ブランチを作成

```bash
git switch -c add-member-[あなたの名前]
```

### 3. プロフィールを追加

`js/members.js` ファイルを開いて、あなたの情報を追加します。

詳しい手順は [MEMBER_GUIDE.md](MEMBER_GUIDE.md) をご覧ください。

### 4. ローカルで確認

`index.html` をブラウザで開いて、あなたのプロフィールカードが表示されているか確認します。

### 5. コミット & プッシュ

```bash
git add js/members.js
git commit -m "add: [あなたの名前]のプロフィールを追加"
git push -u origin add-member-[あなたの名前]
```

### 6. プルリクエストを作成

GitHubのリポジトリページでプルリクエストを作成します。

## 🤝 コントリビューションガイド

### ブランチ命名規則

```
add-member-[あなたの名前]
```

**例:**
- `add-member-yamada`
- `add-member-suzuki`

### コミットメッセージ

```
add: [あなたの名前]のプロフィールを追加
```

### プルリクエスト

- タイトル: `add: [あなたの名前]のプロフィールを追加`
- 説明: 変更内容を簡潔に記載

## ⚠️ 注意事項

- **他のメンバーの情報は編集しないでください**
- **`members` 配列の末尾に追加してください**
- **JavaScriptの記法（カンマ、引用符など）に注意してください**

## 🐛 トラブルシューティング

### プロフィールが表示されない

1. ブラウザのコンソール（F12）を開く
2. エラーメッセージを確認
3. JavaScriptの記法を確認（カンマ、引用符など）

### プッシュできない

1. コラボレーターに追加されているか確認
2. GitHubにログインしているか確認
3. 認証情報が正しいか確認

### コンフリクトが発生した

講師にサポートを依頼してください。
