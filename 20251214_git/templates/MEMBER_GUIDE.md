# メンバー追加ガイド

## 概要

このガイドでは、メンバー紹介サイトに自分のプロフィールを追加する方法を説明します。

---

## Step 1: リポジトリをクローン

```bash
git clone https://github.com/study-basic-hackathon/community-site.git
cd community-site
```

---

## Step 2: ブランチを作成

```bash
git switch -c add-member-[あなたの名前]
```

**例:**
```bash
git switch -c add-member-yamada
```

---

## Step 3: メンバー情報を追加

`js/members.js` ファイルを開いて、`members` 配列にあなたの情報を追加します。

### 追加する場所

```javascript
const members = [
    // 既存のメンバー...
    
    // ⬇️ ここにあなたの情報を追加！
    {
        name: "あなたの名前",
        icon: "😊", // 好きな絵文字を選んでください
        location: "あなたの地域",
        experience: "プログラミング歴",
        language: "好きな言語",
        goal: "今日の目標",
        hobbies: ["趣味1", "趣味2", "趣味3"],
        message: "一言メッセージ"
    },
];
```

### 記入例

```javascript
{
    name: "山田太郎",
    icon: "👨‍💻",
    location: "東京都",
    experience: "1年",
    language: "Python",
    goal: "Gitを使ったチーム開発ができるようになりたい！",
    hobbies: ["プログラミング", "読書", "カフェ巡り"],
    message: "よろしくお願いします！Gitを学んで成長したいです。"
},
```

---

## Step 4: 動作確認（ローカルで表示）

### 方法1: VS Codeの Live Server

1. VS Code で `index.html` を開く
2. 右クリック → 「Open with Live Server」
3. ブラウザで表示を確認

### 方法2: ブラウザで直接開く

1. `index.html` をダブルクリック
2. ブラウザで開く
3. 自分のカードが追加されているか確認

---

## Step 5: コミット

```bash
git add js/members.js
git commit -m "add: [あなたの名前]のプロフィールを追加"
```

**例:**
```bash
git add js/members.js
git commit -m "add: 山田太郎のプロフィールを追加"
```

---

## Step 6: プッシュ

```bash
git push -u origin add-member-[あなたの名前]
```

**例:**
```bash
git push -u origin add-member-yamada
```

---

## Step 7: プルリクエストを作成

1. GitHubのリポジトリページを開く
2. 「Compare & pull request」ボタンをクリック
3. プルリクエストのタイトル:
   ```
   add: [あなたの名前]のプロフィールを追加
   ```
4. 説明欄（オプション）:
   ```
   ## 変更内容
   - メンバー紹介ページに自分のプロフィールを追加しました
   
   ## 確認事項
   - [x] ローカルで表示を確認しました
   - [x] 他のメンバーの情報は変更していません
   ```
5. 「Create pull request」をクリック

## 項目の説明

### name (必須)
- あなたの名前（本名でもニックネームでもOK）
- 例: `"山田太郎"`, `"やまちゃん"`

### icon (必須)
- プロフィールカードに表示される絵文字
- 例: `"👨‍💻"`, `"😊"`, `"🚀"`

### location (必須)
- 住んでいる地域（詳しく書かなくてもOK）
- 例: `"東京都"`, `"関東"`, `"大阪"`

### experience (必須)
- プログラミング歴
- 例: `"1年"`, `"3ヶ月"`, `"これから始める"`

### language (必須)
- 好きな言語・技術
- 例: `"Python"`, `"JavaScript"`, `"HTML/CSS"`

### goal (必須)
- 今日の目標や学びたいこと
- 例: `"Gitを使ったチーム開発ができるようになりたい！"`

### hobbies (必須)
- 趣味や興味があること（配列で指定）
- 例: `["プログラミング", "読書", "ゲーム"]`

### message (必須)
- 一言メッセージ
- 例: `"よろしくお願いします！"`

## よくある質問

### Q: 絵文字が表示されない

**A:** 
- 絵文字をコピー＆ペーストで入力してください
- または絵文字パネルから選択してください

### Q: カンマを忘れてエラーになる

**A:** 
- JavaScriptでは、配列の要素の間に `,` が必要です
- 最後の要素の後ろの `,` はあってもなくてもOK

```javascript
// ❌NG（カンマがない）
{
    name: "太郎"
    icon: "😊"  // ← カンマがない！
}

// ✅OK
{
    name: "太郎",
    icon: "😊"
}
```

### Q: 他の人と同じタイミングで編集したらどうなる？

**A:** 
- 問題ありません！配列の末尾に追加するだけなので、コンフリクトしにくいです
- もしコンフリクトが起きても、講師がサポートします

### Q: ローカルで確認できない

**A:** 
- ブラウザのコンソール（F12キー）を開いてエラーを確認
- JavaScriptの記法が間違っている可能性があります
- 講師に質問してください
